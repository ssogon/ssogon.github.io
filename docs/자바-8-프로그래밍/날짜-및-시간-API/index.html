날짜 및 시간 API

자바 8에서 추가된 java.time 패키지의 날짜, 시간 관련 클래스를 이용하면 기존 Date, Calendar의 문제점들을 해결할 수 있다.

기존 DateTime API의 문제점​

Date​

Date는 클래스 네이밍과는 달리 시간까지 표현할 수 있어 개발자에게 혼동을 준다.

Date date = new Date();
long epochTime = date.getTime();

불변하지 않기 때문에 thread-safe하지 않아 멀티스레드 환경에서 사용하기엔 적합하지 않다.

date.setTime(0);

Calendar​

Calendar는 월이 1이 아닌 0부터 시작하기 때문에 개발자의 주의가 필요하며, 생성자 인자로 int를 받기 때문에 범위를 벗어난 값이 들어갈 수도 있다.

Calendar calendar = new GregorianCalendar(1970, 0, 1);

그리고 Date와 마찬가지로 불변하지 않기 때문에 thread-safe하지 않다.

calendar.add(Calendar.DAY_OF_YEAR, 1);

날짜 및 시간 표현​

Instant​

Instant는 1970-01-01 00:00:00 UTC 부터의 milliseconds를 표현하는 epochTime을 다룬다.

Instant epochTime = Instant.now();
System.out.println(epochTime.getEpochSecond());

현재 지역의 시간을 기준으로 하고 싶다면 ZonedDateTime으로 변환이 필요하다.

ZonedDateTime nowInKorea = epochTime.atZone(ZoneId.of("Asia/Seoul"));

LocalDate, LocalTime, LocalDateTime​

LocalDate, LocalTime, LocalDateTime은 시스템의 로컬 정보를 바탕으로 현재 시간을 가져오는 방법을 제공한다.

LocalDateTime now = LocalDateTime.now();

직접 값을 지정하여 생성할 수도 있다.

LocalDate anniversary = LocalDate.of(1970, Month.JANUARY, 1);

ZonedDateTime​

ZonedDateTime은 특징 지역의 로컬 정보를 바탕으로 현재 시간을 가져오는 방법을 제공한다.

ZonedDateTime nowInKorea = ZonedDateTime.now(ZoneId.of("Asia/Seoul"));

Duration​

Duration은 Instant 사이의 기간을 표현하고자 할 때 사용한다.

Instant now = Instant.now();
Instant after10sec = now.plus(10, ChronoUnit.SECONDS);

Duration duration = Duration.between(now ,after10sec);
System.out.println(duration.getSeconds());

Period​

Period는 LocalDate 사이의 기간을 표현하고자 할 때 사용한다.

LocalDate today = LocalDate.now();
LocalDate thisYearBirthday = LocalDate.of(2020, Month.JANUARY, 1);

Period period = Period.between(today, thisYearBirthday);
System.out.println(period.getDays());

날짜 및 시간 포맷팅​

DateTimeFormatter​

DateTimeFormatter는 날짜 및 시간을 문자열로 포맷팅하는 방법을 제공한다.

LocalDateTime now = LocalDateTime.now();
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("MM-dd-yyyy");
System.out.println(now.format(formatter));

미리 준비되어 있는 포맷을 사용할 수도 있다.

DateTimeFormatter basicIsoDate = DateTimeFormatter.BASIC_ISO_DATE; // yyyyMMdd
DateTimeFormatter isoLocalDate = DateTimeFormatter.ISO_LOCAL_DATE; // yyyy-MM-dd
DateTimeFormatter isoLocalTime = DateTimeFormatter.ISO_LOCAL_TIME; // HH:mm:ss

날짜 및 시간 유틸리티​

TemporalAdjuster​

TemporalAdjuster는 날짜 및 시간을 손쉽게 조정할 수 있도록 도와준다.
TemporalAdjusters에서 자주 사용할만한 TemporalAdjuster들을 제공하고 있다.

LocalDate lastDayOfMonth = LocalDate.now().with(TemporalAdjusters.lastDayOfMonth());
System.out.println(lastDayOfMonth);

원하는게 없다면 직접 만들어 쓰면 된다.

TemporalAdjuster previousWeekday = temporal -> {
    Objects.requireNonNull(temporal, "temporal must not be null");
    LocalDate localDate = LocalDate.from(temporal);
    do {
      localDate = localDate.minusDays(1);
    } while (localDate.getDayOfWeek().getValue() > 5);
    return localDate;
  };

Date-Time API 디자인 원칙​

clear​

클래스 이름은 의도에 맞게 명확히 하여 개발자에게 혼동을 주지 않아야 한다.

fluent​

메서드 체이닝 형태로 API를 제공하여 코드를 읽기 쉽도록 만들어야 한다.

LocalDate payday = LocalDate.now()
    .with(TemporalAdjusters.lastDayOfMonth())
    .minusDays(2);

immutable​

날짜, 시간 클래스는 불변하게 하여 thread-safe함을 보장해야 한다.
만약 연산하는 작업이 필요하다면 새로운 인스턴스를 만들어 반환하게끔 해야 한다.

LocalDate birthday = LocalDate.of(1970, Month.JANUARY, 1);
LocalDate firstBirthday = birthday.plusYears(1);

extensible​

다른 달력 시스템으로 쉽게 확장 가능하도록 만들어야 한다.

JapaneseDate today = JapaneseDate.now();
