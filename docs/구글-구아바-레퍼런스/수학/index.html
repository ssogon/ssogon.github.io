수학

com.google.common.math 패키지에는 다양한 수학 유틸리티들이 포함되어 있다.

Guava: Math

개요​

수학 유틸리티는 숫자 유형에 따라 크게 IntMath, LongMath, DoubleMath, BigIntegerMath로 나누어져 있다.

int longFloor = LongMath.log2(n, FLOOR);
int mustNotOverflow = IntMath.checkedMultiply(x, y);
long quotient = LongMath.divide(knownMultipleOfThree, 3, RoundingMode.UNNECESSARY); // fail fast on non-multiple of 3
BigInteger nearestInteger = DoubleMath.roundToBigInteger(d, RoundingMode.HALF_EVEN);
BigInteger sideLength = BigIntegerMath.sqrt(area, CEILING);

추가로 LinearTransformation은 y = max + b와 같은 double 값 사이의 선형 변환을 나타낸다.

LinearTransformation.mapping(1, 1)
    .and(5, 6)
    .slope(); // 1.25

이들 보다 덜 수학적인 함수들은 com.google.common.primitives 패키지에서 찾아 볼 수 있다.

Guava: Contents

사용상의 이점​

  • 이 유틸리티들은 비정상적인 오버플로우에 대해 철처하게 테스트되었다.
    만약 전제조건이 어긋나면 빠르게 실패하는 fail-fast 방식을 취한다.
  • Apache Commons의 MathUtils과 대체로 비슷한 성능을 보이며 경우에 따라 더 빠른 모습을 보인다.
  • 읽기 쉽고 올바른 프로그래밍 습관을 장려하도록 설계되었다.

정수​

int, long, BigInteger 타입을 다루는 유틸리티에는 IntMath, LongMath, BigIntegerMath가 있다.

IntMath.checkedAdd(Integer.MAX_VALUE, Integer.MAX_VALUE); // throws ArithmeticException

반올림​

IntMath, LongMath, BigIntegerMath는 다양한 산술 연산을 지원한다.

연산IntMathLongMathBigIntegerMath
나누기int divide(int, int, RoundingMode)long divide(long, long, RoundingMode)BigInteger divide(BigInteger, BigInteger, RoundingMode)
밑이 2인 로그int log2(int, RoundingMode)long log2(long, RoundingMode)BigInteger log2(BigInteger, RoundingMode)
밑이 10인 로그int log10(int, RoundingMode)long log10(long, RoundingMode)BigInteger log10(BigInteger, RoundingMode)
제곱근int sqrt(int, RoundingMode)long sqrt(long, RoundingMode)BigInteger sqrt(BigInteger, RoundingMode)

다음은 반올림 방식을 지정하는 java.math.RoundingMode에 대한 설명이다.

  • DOWN: 0과 가까워지도록 반올림한다.
  • UP: 0과 멀어지도록 반올림한다.
  • FLOOR: 음의 방향으로 반올림한다.
  • CEILING: 양의 방향으로 반올림한다.
  • HALF_UP: 가장 가까운 쪽으로 반올림하고, x.5는 올린다.
  • HALF_DOWN: 가장 가까운 쪽으로 반올림하고, x.5는 내린다.
  • HALF_EVEN: 가장 가까운 쪽으로 반올림하고, x.5는 짝수를 향해 반올림한다.

연산IntMathLongMathBigIntegerMath

  • 나머지는 항상 양수다. (-5 mod 3 is 1)
  • 지수는 오버플로우될 수 있다.
  • 팩토리얼과 이항 계수는 입력이 너무 큰 경우 MAX_VALUE를 반환한다.

Guava: Additional functions

부동소수점 연산​

DoubleMath에는 부동소수점을 위한 몇 가지 유용한 메서드들이 있다.

메서드설명
boolean isMathematicalInteger(double)입력이 유한하고 정확한 정수인지 확인한다.
int roundToInt(double, RoundingMode)지정된 숫자를 반올림하고 int에 딱 맞으면 int로 캐스팅한다.
그렇지 않은 경우 ArithmeticException을 발생시킨다.
long roundToLong(double, RoundingMode)지정된 숫자를 반올림하고 long에 딱 맞으면 long로 캐스팅한다.
그렇지 않은 경우 ArithmeticException을 발생시킨다.
BigInteger roundToBigInteger(double, RoundingMode)지정된 숫자를 반올림하고 BigInteger에 딱 맞으면 BigInteger로 캐스팅한다.
그렇지 않은 경우 ArithmeticException을 발생시킨다.
int log2(double, RoundingMode)밑이 2인 로그를 취하고 반올림한다.
