프리미티브 타입

primitive 타입을 지원하는 guava의 primitive 유틸리티에 대해 설명한다.

개요​

이 글을 읽기 전에 찾으려는 것이 Arrays 또는 JDK 래퍼 타입에 있는 지 먼저 확인해야 한다.

다음은 com.google.common.primitives 패키지가 제공하는 기능들이다.

  • primitive 배열과 Collection간의 인터페이스
  • primitive 타입의 byte 배열 변환
  • 부호없는(unsigned) 변수 지원

byteBytes, SignedBytes, UnsignedBytes
shortShorts
intInts, UnsingedInteger, UnsignedInts

Guava: Primitives Overview

프리미티브 배열 메서드​

primitive 배열은 primitive 타입을 집계하는 방법 중 가장 효율적인 방법이다. (메모리 및 성능 측면에서)

정적 메서드설명유사한 메서드
List<Wrapper> asList(prim...)primitive 배열을 래퍼 타입의 List로 래핑한다.Arrays.asList(T...)
prim[] toArray(Collection<Wrapper>)Collection을 primitive[]로 복사하여 반환한다.collection.toArray(T[])
prim[] concat(prim[]...)여러 primitive 배열들을 병합한다.Iterables.concat(Iterable<T>...)
boolean contains(prim[], prim)특정 요소가 primitive 배열 안에 있는지 확인한다.collection.contains(Object)
int indexOf(prim[], prim)primitive 배열에서 특정 요소가 첫 번째로 등장하는 인덱스를 반환한다.
위치를 찾지 못했다면 -1을 반환한다.		list.indexOf(Object)
int lastIndexOf(prim[], prim)primitive 배열에서 특정 요소가 마지막으로 등장하는 인덱스를 반환한다.
위치를 찾지 못했다면 -1을 반환한다.		list.lastIndexOf(Object)
prim min(prim...)primitive 배열에서 가장 작은 요소를 반환한다.Collections.min(Collection<T>)
prim max(prim...)primitive 배열에서 가장 큰 요소를 반환한다.Collections.max(Collection<T>)
String join(String, prim)구분자로 primitive 배열의 요소들을 결합하여 반환한다.Joiner.on(String).join(Collection)
Comparator<prim[]> lexicographicalComparator()사전 순으로 primitive 배열을 비교하는 비교하는 Comparator<prim[]>을 반환한다.Ordering.natural().lexicographical()

캐스팅 유틸리티 메서드​

다음은 캐스팅과 관련된 메서드다.

정적 메서드설명
prim checkedCast(long)특정 값을 prim으로 캐스팅한다.
지정된 값이 prim에 맞지 않는 경우 IllegalArgumentException이 발생한다.
prim saturatedCast(long)특정 값을 prim으로 캐스팅한다.
지정된 값이 prim에 맞지 않는 경우 가장 근사한 prim 값이 반환된다.

참고로 반올림은 DoubleMath에서 제공하고 있다.

DoubleMath.roundToInt(1.5, RoundingMode.DOWN); // 1

Guava: General utility methods

바이트 변환 메서드​

primitive 타입과 빅-엔디안 byte 배열 간의 변환을 지원하는 메서드도 제공한다.

정적 메서드설명
prim fromByteArray(byte[])빅-엔디안 byte 배열을 primitive 값으로 변환한다.
byte 배열의 길이가 Prims.BYTES보다 작을 경우 IllegalArgumentException이 발생한다.
prim fromBytes(byte...)Prims.BYTES만큼의 byte들을 primitive 값으로 변환한다.
byte[] toByteArray(prim)primitive 값을 빅-엔디안 byte 배열로 변환한다.

primitive 값을 표현하는데 필요한 바이트 수를 나타내는 상수도 제공하고 있다.

System.out.println(Ints.BYTES); // 4

부호없는 변수 지원​

문자열 변환 메서드​

유틸리티 클래스인 UnsignedInts와 UnsignedLongs에는 부호없는 변수와 문자열 간의 변환을 지원하는 메서드들이 포함되어 있다.
해당 유틸리티에 부호없는 값만 전달되는지 확인하는 것은 사용자의 책임이다.

정적 메서드설명
int UnsignedInts.parseUnsignedInt(String)
long UnsignedLongs.parseUnsignedLong(String)		10진수 문자열을 파싱하여 부호없는 값을 반환한다.
int UnsignedInts.parseUnsignedInt(String, int)
Long UnsignedLongs.parseUnsignedLong(String, int)		특정 진법을 가지는 문자열을 파싱하여 부호없는 값을 반환한다.
String UnsignedInts.toString(int)
String UnsignedLongs.toString(long)		특정 부호없는 값을 10진수 문자열로 나타낸다.
String UnsignedInts.toString(int, int)
String UnsignedLongs.toString(long, int)		특정 부호없는 값을 특정 진법을 가지는 문자열로 나타낸다.

Guava: Generic utilities

래퍼​

부호없는 래퍼 타입인 UnsignedInt와 UnsignedLong는 몇 가지 유용한 메서드들을 제공하고 있다.

정적 메서드설명
UnsignedPrim valueOf(BigInteger)BigInteger로부터 부호없는 래퍼 타입을 반환한다.
만약 BigInteger가 음수이거나 래퍼 타입에 맞지 않는다면 IllegalArgumentException이 발생한다.
UnsignedPrim valueOf(long)long으로부터 부호없는 래퍼 타입을 반환한다.
만약 long이 음수이거나 래퍼 타입에 맞지 않는다면 IllegalArgumentException이 발생한다.
UnsignedPrim fromPrimBits(prim)특정 값으로부터 부호없는 래퍼 타입을 반환한다.
예를 들어, UnsignedInteger.fromIntBits(1 << 31)은 1 << 31이 음수임에도 불구하고 2의 31승을 나타낸다

인스턴스 메서드설명
UnsignedPrim plus(UnsignedPrim)
UnsignedPrim minus(UnsignedPrim)
UnsignedPrim times(UnsignedPrim)
UnsignedPrim divideBy(UnsignedPrim)
UnsignedPrim mod(UnsignedPrim)		부호없는 타입 간의 사칙연산을 수행한다.
BigInteger bigIntegerValue()현재 부호없는 래퍼 타입의 값을 BigInterger로 반환한다.
String toString()현재 부호없는 래퍼 타입의 값을 String으로 반환한다.
